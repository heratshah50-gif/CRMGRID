# CRM Basegrid Landing Page

Next.js landing page for CRM Basegrid.

## Setup

```bash
cd landing-app
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Tech

- **Next.js 13** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Plus Jakarta Sans** (Google Fonts)
